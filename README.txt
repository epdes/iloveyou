Love Letter Website
===================

Open index.html in a browser to preview the romantic book website.

This version already includes:

- The full Malayalam/English letter to Rijo.
- The three provided photos in assets/photo-1.jpg, assets/photo-2.jpg, and assets/photo-3.png.
- The provided background music in assets/thousand-years.mp3.

To make small changes:

1. Edit script.js.
   - Change yourName, title, subtitle, photo captions, or letter pages.

2. Replace media if needed.
   - Music: assets/thousand-years.mp3
   - Photos:
     assets/photo-1.jpg
     assets/photo-2.jpg
     assets/photo-3.png

3. Keep file names the same, or update the paths in script.js.

Controls:

- Click the round play button to start or pause the music.
- Click the speaker button to mute or unmute.
- Use the side arrows, mobile buttons, keyboard arrows, or swipe left/right on a phone to turn pages.
